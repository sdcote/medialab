# Workflows
This is my personal workflows for a few common tasks.

## iPhone Media
1. ImageCapture to the Phone folder
1. `snap FileClassifier` to move IMG_\d files to either Images or Videos
1. Check remaining files...these are probably image downloads from other apps
1. `snap MDCimages` places all the images into date sub-folders
1. Check the images directory to make sure everything has been classified
1. move all folders to the Pictures directory (Use Meld if necessary)
1. `snap MDCvideos` places all the videos into date sub-folders
1. Check the videos directory to make sure everything has been classified
1. move all folders to the Videos directory (Use Meld if necessary)
